# Bex Sigma Loan - Premium Fintech Landing Page

A cinematic React + Vite loan website with dark theme, floating financial particles, animated sections, and a premium fintech look.

## Run locally

1. Open terminal in `c:\Users\Dell\OneDrive\Desktop\Business website loan`
2. Run `npm install`
3. Run `npm run dev`
4. Open the local URL shown by Vite

## Features

- Full-screen animated hero with collage imagery
- Animated money and currency particle background
- Loan types with glow hover effects
- Glassmorphism EMI calculator with live value updates
- Service grid, process flow, testimonials, and CTA sections
- Contact form with email integration and toast notifications
- Multi-step loan application modal with toast notifications
- Responsive layout for desktop and mobile

## Email Setup

Form submissions (contact form and loan applications) are sent to `bexsigmatech@gmail.com` using EmailJS.

**Setup Required:**
1. Create a free EmailJS account at [emailjs.com](https://www.emailjs.com/)
2. Configure email service and templates
3. Update the `.env` file with your EmailJS credentials

See `EMAIL_SETUP.md` for detailed instructions.
